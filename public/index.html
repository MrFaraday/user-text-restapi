POST /auth - выдача токена
request body:
{
  "user": [username],
  "password": [password]
}
response:
{
  "access_token": [token],
  "refreshToken": [refresh_token]
}
  
POST /refresh
request bodyL
{
  "refreshToken": [refresh_token]
}
Ответ такой же, как и при аутентификации
  
Дальнейшие запросы должны идти с токеном в заголовке или теле запроса
  
GET /[user] - имя пользователя для просмотра
  
GET /users/[page?] - просмотр всех пользователей с их записями, пагинация 3 на страницу
